## 🚀 项目简介

**FurryArtStudio** 是一款用 VB.NET 开发的桌面应用，致力于帮助 Furry 社区用户高效整理角色设定、绘画素材、参考图集等内容 🖼️。  
它支持直观的分类、标签、搜索与批量操作，是艺术创作者和资料收集者的好帮手 🎨

---

## 📦 功能亮点

- 📁 管理手稿、图片、参考资料等多种类型内容  
- 🔍 支持文本搜索和标签过滤  
- 📌 清晰的结构设计  
- 🐶 面向 Furry 创作者的专用管理工具  
- 🧠 轻量、开源、可扩展

---

## 🛠️ 快速开始

1. 克隆仓库：

```bash
git clone https://github.com/xionglongztz/FurryArtStudio.git
```

2. 使用 Visual Studio 打开解决方案：

```bash
FurryArtStudio.sln
```

3. 构建并运行 🎉
